Media query variables

Included in the mixins are a couple of useful variables for use with media queries.

For starters there are two general, @mobile and @tablet.

@media @mobile {
  // Mobile adjustments
}

If you need to pinpoint a specific orientation just add either @portrait or @landscape to the media query.

@media @tablet @portrait {
  // Tablet portrait adjustments
}

There's also two device specific ones for @iphone4 and @iphone5 if you want to be really precise.

Appearance

Default values and arguments

  • @prop: none

Set the appearance of the element. Takes one arguments for the property. Does not affect select dropdowns in Firefox.

.appearance();
↓

Border radius

Default values and arguments

  • @radius: 3px

Adjust the border radius of an element. Can also be used to make circle shapes.

.border-radius {
  .border-radius(100px);
  .size(100px);
}

Box shadow

Set the outer box shadow of an element.

.box-shadow {
  .box-shadow(0 0 20px rgba(33,33,33,0.6));
}

Box sizing

Default values and arguments

  • @type: border-box

Change the box model of an element. Defaults to border-box, which if used with padding doesn't expand the box in size.

.box-sizing {
  .box-sizing();
}

Center

Default values and arguments

  • @coord: no default
  • @direction: top (for x) / left (for y)
  • @y, @x: 0

Centers an elements within it's relative parent. Example two shows how the @direction and @x values can be manipulated.

// Center X
.center(x);

// Start from the right instead of default left and move
// element 40px from the right
.center(y, right, 40px); 

// Center in middle
.center(both);
Centered on x-axis
Centered on y-axis but 40 pixels from the right
Centered on both axes

Columns

Default values and arguments

  • @columns: 2
  • @gap: 20px

Split contents of an element into columns.

.columns(3, 10px);

Font

Default values and arguments

  • @size: no default
  • @style: normal

Sets font-size in rem with and a fallback in px.

.font(40px, italic);
Hello World!

Inner shadow

Default values and arguments

  • @shadow: 1px 1px 0px rgba(0,0,0,1)

Adds an inner shadow to an element.

.inner-shadow();

Opacity

Default values and arguments

  • @opacity: no default

Set the opacity of an an element and its contents. Also adds a filter alpha for older versions of IE.

.opacity(30);
Hello

Position

Default values and arguments

  • @direction1: no default
  • @direction2: no default
  • @position1: 0
  • @position2: 0
  • @positioning: absolute
  • @full: false

Set the position of an item. In the first example it defaults to being positioned absolute, but as a fifth parameter you can specify any positioning.

.position(top,right,20px,20px);
I'm hanging out in the corner

Another useful feature is the sixth parameter which if set to true makes the element cover the whole parent.

.position(top,left,0,0,absolute,true);
I'm covering EVERYTHING

Prefix

Default values and arguments

  • @property: no default
  • @value: no default

Return any property with browser specific prefixes.

.prefix(box-sizing, border-box);

Returns

-webkit-box-sizing: border-box;
   -moz-box-sizing: border-box;
    -ms-box-sizing: border-box;
     -o-box-sizing: border-box;
        box-sizing: border-box;

Rotate

Default values and arguments

  • @angle: no default

Rotate an element a given amount of degrees or pixels

.rotate(-45deg);

Scale

Default values and arguments

  • @ratio: no default

Scale an element

.scale(2);

Text overflow

Puts all text of an element on one line and adds an ellipsis if the length of the text is wider than the element.

.text-overflow();

Returns

overflow: hidden;
text-overflow: ellipsis;
white-space: nowrap;

Transform

Default values and arguments

  • @transform: no default

Set a transform of the element for anything you like.

.transform(translate3d(0,0,0));

Returns

-webkit-transform: translate3d(0,0,0);
   -moz-transform: translate3d(0,0,0);
    -ms-transform: translate3d(0,0,0);
     -o-transform: translate3d(0,0,0);
        transform: translate3d(0,0,0);

Transition

Default values and arguments

  • @type: all
  • @time: 300ms
  • @animate: ease

Set a transition to the element

.transition(color, 150ms, linear);

Returns

-webkit-transition: color 150ms linear;
   -moz-transition: color 150ms linear;
    -ms-transition: color 150ms linear;
     -o-transition: color 150ms linear;
        transition: color 150ms linear;

Triangle

Default values and arguments

  • @direction: no default
  • @size: no default
  • @color: @default-bg

Makes a triangle

.triangle(up, 50px, rgb(243, 92, 159));
.triangle(right, 20px, rgb(243, 92, 159));
.triangle(down, 100px, rgb(243, 92, 159));
.triangle(left, 40px, rgb(243, 92, 159));

A bonus one for the triangle is to point it in an angle to get it in to corners

Default values and arguments

  • @direction1: no default
  • @direction2: no default
  • @size: no default
  • @color: @default-bg
.triangle-corner(top, right, 50px, rgba(243, 92, 159));
.triangle-corner(bottom, right, 50px, rgba(243, 92, 159));
.triangle-corner(bottom, left, 50px, rgba(243, 92, 159));
.triangle-corner(top, left, 50px, rgba(243, 92, 159));

User select

Default values and arguments

  • @arg: none

On what level should an element be selectable.

.user-select();

Returns

-webkit-user-select: none;
   -moz-user-select: none;
    -ms-user-select: none;
     -o-user-select: none;
        user-select: none;
